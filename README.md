# Wishlist Price Tracker

A Python project for tracking products from multiple retail websites, extracting product information from URLs, matching similar items, and identifying the cheapest available option.

## Project Goals

- Read product links from a CSV file
- Scrape product name, price, image, and store
- Normalize product names for comparison
- Match identical or similar products across stores
- Show the cheapest option for each product
- Track price changes over time

## Tech Stack

- Python
- Pandas
- Requests
- BeautifulSoup
- RapidFuzz
- SQLite eventually
